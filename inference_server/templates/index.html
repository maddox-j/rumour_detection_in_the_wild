Twitter Rumour Detector © Server 2023

Welcome to the server for the Twitter rumour detection browser extension that I have created during my 4th year dissertation at the University of Edinburgh.

This server allows users to interact with the state-of-the-art rumour detection models and classify their tweets.

Privacy Notice

This server, and its assosicated browser extension, are for testing purposes only. They aim to show a proof-of-concept application in support of my undergraduate dissertation. As such, no personal details whatsover are stored nor handled when interacting with this browser extension.

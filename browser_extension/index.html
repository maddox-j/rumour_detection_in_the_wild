Twitter Rumour Detector © 2023

Welcome to this Twitter rumour detection browser extension.

Using this browser extension is pretty self-explanatory. Navigate your way to a Tweet you wish to detect. Click the Detect rumour button, and hey, presto, it works! Your Tweet will be classified, and relevant news articles will be supplied below.

NB: Due to the way the Twiter Developer API works, we can only detect rumours for tweets in the last seven days. Otherwise, we will recieve an error.
